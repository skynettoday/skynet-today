#### [Google‚Äôs Gemini to power Apple‚Äôs AI features like Siri](https://techcrunch.com/2026/01/12/googles-gemini-to-power-apples-ai-features-like-siri/)
![](https://techcrunch.com/wp-content/uploads/2025/01/apple-intelligence-iphone-mac.jpg?resize=1200,800)

Apple and Google confirmed a multi-year partnership to use Google‚Äôs Gemini models and cloud for Apple‚Äôs AI features, including Siri and future Apple foundational models. Anthropic launched Claude for Healthcare, a suite of tools for providers, payers, and patients that competes with OpenAI‚Äôs recently announced ChatGPT Health. Anthropic emphasizes ‚Äúagent skills‚Äù and ‚Äúconnectors‚Äù that integrate Claude with healthcare systems and databases, including CMS‚Äôs Coverage Database, ICD-10, the National Provider Identifier Standard, and PubMed. #### [Anthropic‚Äôs new Cowork tool offers Claude Code without the code](https://techcrunch.com/2026/01/12/anthropics-new-cowork-tool-offers-claude-code-without-the-code/)
![](https://techcrunch.com/wp-content/uploads/2025/08/Claude-Chrome-Ext_email-hero-hero.png?w=1200)

Anthropic announced Cowork, a new tool inside the Claude Desktop app that brings Claude Code‚Äôs agentic capabilities to non-technical users. Users grant Cowork access to a specific folder, and then direct it via the standard chat interface to read, modify, and create files within that sandbox. Built on the Claude Agent SDK and the same underlying model as Claude Code, Cowork removes the need for command-line tools or virtual environments while preserving multi-step autonomous actions. It‚Äôs in research preview, initially for Max subscribers, with a waitlist for others.

Targeted at general-purpose workflows early Claude Code users were already attempting, Cowork can assemble expense reports from receipt photos, organize and tag media files, scan social media posts, or analyze conversation logs. Like Claude Code, it can execute chains of actions without user input, which raises risks if instructions are vague or conflicting; Anthropic warns about prompt injection and accidental file deletion and urges clear, unambiguous prompts. The folder-partitioned sandbox helps manage data access but doesn‚Äôt eliminate misuse risks. Cowork follows rapid expansion of Claude Code interfaces: the original CLI launched in November 2024, with a web UI in October and a Slack app two months later.

[GenCtrl -- A Formal Controllability Toolkit for Generative Models](https://arxiv.org/abs/2601.05637). This paper frames controlled generation as a control-theoretic problem, provides PAC Monte Carlo algorithms to estimate reachable and controllable output sets for opaque generative models across discrete and continuous spaces, and presents empirical results showing controllability varies widely across models and tasks.

[RealMem: Benchmarking LLMs in Real-World Memory-Driven Interaction](https://arxiv.org/abs/2601.06966). The authors introduce RealMem, a 2,000+ cross-session benchmark and a three-stage synthesis pipeline to evaluate how well agent memory systems retrieve, compress, and use dynamically evolving, project-centric memories‚Äîfinding current methods struggle on precision, long-range coherence, and latency/resource trade-offs.

[Beyond Hard Masks: Progressive Token Evolution for Diffusion Language Models](https://arxiv.org/abs/2601.07351). Their method represents tokens as evolving soft probability distributions throughout diffusion, uses continuous trajectory supervision to train progressive refinement, and remains compatible with KV-caching and blockwise diffusion while improving performance over standard masked diffusion baselines.

[Extracting books from production language models](https://arxiv.org/abs/2601.02671). The authors show that a simple two-phase prompting and, when needed, jailbreaking strategy can sometimes extract long, near‚Äëverbatim portions of copyrighted books from multiple production LLMs, with success and cost varying widely by model and configuration.

[PaCoRe: Learning to Scale Test-Time Compute with Parallel Coordinated Reasoning](https://arxiv.org/abs/2601.05593). PaCoRe coordinates many parallel reasoning trajectories, compresses their findings into compact messages, and uses outcome-based reinforcement learning to train models to synthesize those messages into superior final solutions, enabling much larger effective test-time compute without expanding the context window.
